Bug Tracker

This application is a bug tracking system that lets you keep track of software bugs in development projects. The software allows for users to submit tickets and those tickets are tracked through status from Open to Complete. A role based security system was implemented to create permission based roles. The roles of Admin, Project Manager, Developer and Submitter restrict what the user can see and do.

The app was built with ASP.NET Core, MVC, C# and SQL. The front end was developed with HTML, JavaScript and Bootstrap. The database was built with SQL Server and hosted on Azure.

Sorting Algorithms Visualizer

I built this application with the intent of improving my JavaScript skills and learning classic sorting algorithms. The app has a simple user interface where you can generate a new array of numbers, change the array size and sorting speed, select the algorithm that you want to see and finally a sorting button to start the process.

This app was built using JavaScript, HTML, CSS, Bootstrap and provides visualization for Merge Sort, Quick Sort, Insertion Sort, Selection Sort and Bubble Sort

Secure Chat App

In this project I created an app for secure communication, using a client-server architecture. The programming language used for this project is C# and in order to create the app, I used different .NET frameworks, such as WCF, WPF and EF.

The communication between server and client is of type TCP full duplex and it is secured using AES (Advanced Enryption Standard). For the key exchange between clients I used Diffie-Hellman with elliptic curves, more precisely Curve25519.
